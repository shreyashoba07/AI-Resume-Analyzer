# AI-Powered Resume Analyzer and Job Recommendation System

## Project Overview

The AI-Powered Resume Analyzer and Job Recommendation System is a web-based application that helps users upload resumes, analyze skills, generate resume scores, and receive job recommendations.

## Features

* User Registration
* User Login
* Dashboard
* Resume Upload
* Resume Analysis
* Resume Scoring
* Skill Identification
* Job Recommendations

## Technologies Used

* HTML
* CSS
* PHP
* MySQL
* XAMPP
* phpMyAdmin

## Project Workflow

1. User Registration
2. User Login
3. Access Dashboard
4. Upload Resume
5. Analyze Resume
6. View Resume Score and Job Recommendations

## Future Enhancements

* Natural Language Processing (NLP)
* Machine Learning-based Resume Parsing
* ATS Score Prediction
* Personalized Career Suggestions
